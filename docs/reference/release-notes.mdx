## ox release-notes

Display release notes for the current version

### Synopsis

Display formatted release notes showing what's new, changed, and fixed in the current version.

```
ox release-notes [flags]
```

### Options

```
  -h, --help     help for release-notes
      --latest   show only the latest version (default: false)
      --raw      output raw markdown without formatting (default: false)
```
