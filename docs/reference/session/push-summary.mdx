## ox session push-summary

Push session summary to ledger

### Synopsis

Push a generated summary JSON file to the session's ledger directory.

Called automatically by coding agents after processing the summary_prompt
from 'ox session stop'. The summary file is committed directly to git
(no LFS needed) alongside meta.json.

```
ox session push-summary [flags]
```

### Options

```
      --file string          Path to the summary JSON file
  -h, --help                 help for push-summary
      --session-dir string   Path to the session directory in the ledger
```
