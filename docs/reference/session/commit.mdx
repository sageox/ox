## ox session commit

Commit pending sessions

### Synopsis

Commit pending session changes to the ledger.

This commits any sessions that haven't been committed yet.
Use 'ox sync' to also push to remote.

```
ox session commit [flags]
```

  -m, --message string   Custom commit message
