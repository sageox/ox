## ox daemon start

Start the background daemon

### Synopsis

Starts the SageOx daemon in the background for automatic ledger sync.

```
ox daemon start [flags]
```

### Options

```
      --foreground   run in foreground (for debugging)
  -h, --help         help for start
```
