## ox daemon kill

Kill daemon(s)

### Synopsis

Kill one or more ox daemons. Use --all to kill all running daemons.

```
ox daemon kill [flags]
```

### Options

```
      --all    kill all running ox daemons
  -h, --help   help for kill
```
