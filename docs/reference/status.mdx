## ox status

Display SageOx status and directory locations

### Synopsis

Display current authentication status, configuration, and data locations.

Shows authentication state, project initialization, ledger/team context sync status,
daemon health, and a tree view of all SageOx directory locations.

```
ox status [flags]
```
