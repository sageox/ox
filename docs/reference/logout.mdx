## ox logout

Log out of SageOx

### Synopsis

Remove local authentication token and log out of SageOx.

```
ox logout [flags]
```

### Options

```
      --all               logout from all endpoints
      --endpoint string   endpoint to logout from
  -f, --force             skip confirmation prompt (for scripting)
  -h, --help              help for logout
```
