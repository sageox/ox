## ox agent redact test

Test redaction on sample text

### Synopsis

Apply all redaction rules (built-in + custom) to sample text.

Pass text as an argument, or use '-' to read from stdin.

Examples:
  ox agent redact test "AKIA1234567890123456"
  echo "password='hunter2'" | ox agent redact test -

```
ox agent redact test [text|-] [flags]
```
